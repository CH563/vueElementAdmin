# 后台管理系统

> 后台管理系统模版

## Build Setup

``` bash
# 所在安装整个项目依赖
npm install

# 开启热更新调试，访问地址localhost:8080
npm run dev

# 打包压缩发布生产
npm run build

# 打包压缩发布生产，而且生成报告
npm run build --report
```
